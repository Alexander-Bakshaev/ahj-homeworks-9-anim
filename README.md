
# Collapse Widget

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://alexander-bakshaev.github.io/ahj-homeworks-9-anim/)

## Описание

Collapse Widget - это интерактивный компонент, который позволяет сворачивать и разворачивать контент. Простое и удобное решение для отображения дополнительной информации на странице.

## Особенности

- Анимация плавного появления/скрытия контента
- Адаптивный дизайн
- Простота использования
- Легкая интеграция в любой проект

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/ahj-homeworks-9-anim.git
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите в режиме разработки:
   ```bash
   npm start
   ```
4. Для продакшн-сборки:
   ```bash
   npm run build
   ```

## Использование

1. Добавьте HTML-разметку:
   ```html
   <div class="container">
     <button id="collapse-button" class="collapse-button">Collapse</button>
     <div id="collapsible-content" class="collapsible-content">
       <div class="content-text">
         Ваш текст здесь...
       </div>
     </div>
   </div>
   ```
